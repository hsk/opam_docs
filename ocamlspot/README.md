# ocamlspot

ocamlspotは関数を検索するコマンドです。

## インストール

	$ opam install ocamlspot

とすると、

	opam install ocamlspot
	[ERROR] ocamlspot is not available because it requires OCaml = 4.01.0.

と怒られてしまいます。

	$ opam switch 4.01.0

としてバージョンを4.01.0にしましょう。

## 使い方

	$ ocamlspot
